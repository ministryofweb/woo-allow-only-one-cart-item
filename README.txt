This WordPress plugin ensures that only one item can be in the WooCommerce cart at any given time.

== Description ==

Often it's required to limit the cart to one item. This plugin achieves this by emptying the cart before
a new item is added.

== Requirements ==

- PHP 5.6
- WooCommerce 3.x

== Installation ==

Installing WP Subtitle Support for Better Internal Link Search is just like installing most other plugins.

[Check out the codex](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins) if you have any questions.

This plugin doesn't provide any options or settings.
